# FruitNinja AI

Applying Vision ML techniques to the Fruit Ninja game with few labels.

## Execução do bot

Instale o projeto com:

Execute via entrypoint:

```bash
fruitnai-bot
```

Ou execute diretamente pelo módulo:
Inferência padrão (sem debug/logs):

```bash
python -m bot.run
```

Tuning com diagnósticos (draw/debug/logs):

```bash
python -m bot.run --enable-diagnostics
```
